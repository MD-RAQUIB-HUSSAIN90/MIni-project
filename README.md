# Real-Time Search Filter

A modern and responsive search filter application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

This project displays user profile cards dynamically and filters them instantly as the user types in the search bar. If no matching user is found, a **"User Not Found"** message is displayed.

## 🚀 Features

- 🔍 Real-time search filtering
- 👤 Dynamic profile cards
- 🎨 Modern and clean UI
- 🌫️ Blurred background effect
- 📱 Fully responsive design
- ⚡ Fast and lightweight
- ❌ Displays "User Not Found" for invalid searches
- 💻 Built using Vanilla JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
